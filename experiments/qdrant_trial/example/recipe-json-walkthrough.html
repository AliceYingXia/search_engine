flow.description Flow model attribute (DB column) Optional recipe description (often null)

V2 _dp() Processing Pipeline (engine-runner, Go)

1. Tokenize
State machine detects #{_dp(' ... ')}
text_mode_tokenizer.go:10-15
2. Parse
JSON → datapill struct (pill_type, provider, line, path)
text_mode_parser.go:33-56 → data_pill.go:38
3. Validate
pill_type must be "output". provider & line required.
data_pill.go:74-88
4. Resolve

text_mode_renderer.go:170-217
5. Return Value
Single pill → typed value. Mixed → string concat.
i
Single pill optimization: If the expression is a single pill with no surrounding text, the raw typed value (int, float, bool, array, object) is preserved. If mixed with text, everything is coerced to strings and concatenated.

Special Path Elements (V2)

____source Iteration Pattern

When an input object has a ____source key, engine-runner iterates the source array:

1
